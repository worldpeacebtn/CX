## What this is
Static React site (Vite) for the X42 public brief and lawyer intake. Dark cyber aesthetic. Includes a Three.js background and inline SVG logo.

## How to deploy (quick)
1. Create a GitHub repo and push these files (use GitHub web editor on mobile).  
2. Connect the repo to Vercel (vercel.com -> import project) and deploy.  
3. Vercel auto-builds with `npm run build`.

## Notes
- Do not publish serial numbers or full personal data publicly. Provide details to lawyers only via secure channels.
- To edit content, update files in `src/data/*` and redeploy.
